# Using this responsibly

This is an **OWASP LLM Top 10 security reference**, built so people can
**self-learn AI fluency and LLM security**. The real engagement here is *learning*
— not reselling. If that's why you're here, welcome.

## You're welcome to
- Clone it and run it locally
- Read the write-ups and code, learn from them, adapt them for your own study
- Share and build on them — under the same non-commercial terms, with credit

## Please don't
- Sell it, or fold the content/code into a commercial product, course, or paid service
- Strip attribution / the `Required Notice`
- Republish the write-ups as your own

## Clone & run
```sh
git clone https://github.com/anandsureshworks/owasp-llm-top10.git
cd owasp-llm-top10
npm install
npm run dev        # http://localhost:3000
```
